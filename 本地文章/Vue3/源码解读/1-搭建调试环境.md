### 安装依赖

vue-next使用的是 pnpm 进行包管理，所以需要先全局安装一下 pnpm

```bash
npm i pnpm -g
```

删掉package.json中的两个包（用不到，而且太大了）

```js
puppeteer
@types/puppeteer
```

然后在项目根目录安装依赖

```bash
pnpm i
```



### 修改脚本

package.json

```json
"dev": "node scripts/dev.js --sourcemap"
```



### 打包

```bash
pnpm run dev
```
